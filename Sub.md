**以下订阅链接均为秋风广告规则正式版本的订阅链接，其中：**

- Github Raw：

Github 官方的文件直链，访问速度最快，可靠性最高。

*PS: 对中国大陆用户而言，此订阅链接可能会被防火墙阻断，若无法使用，请使用其它订阅。*

- 渺软CDN反代：

中国大陆备案，访问速度快，7~10天更新一次，可靠性较高。

- jsDelivr：

jsDriver官方的加速订阅，访问速度快，一周左右更新一次，可靠性较高。

 - ghproxy反代：

为 Github raw 的反代，更新速度较快，但可靠性一般。
