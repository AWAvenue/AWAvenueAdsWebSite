# 訂閱規則

## 訂閱須知

::: danger 注意
**在訂閱這些規則之前，請完整閱讀[用戶協議](./Protocol.md)，並確保您有處理潛在問題的能力。**
:::

**以下是秋風廣告規則正式版本的訂閱鏈接，包括:**

- Github Raw：

Github 官方文件的直鏈，訪問速度最快，可靠性最高。

- 微軟 CDN 反代：

中國大陸備案，訪問速度快，7~10天更新一次，可靠性較高。

- jsDelivr：

jsDriver官方的加速訂閱，訪問速度快，一周左右更新一次，可靠性較高。

 - ghproxy 反代：

為 Github raw 的反代，更新速度較快，但可靠性一般。
