## 如何为本仓库贡献？

环境准备：安装 [Node.js](https://nodejs.org/) 18.0或以上版本，根据需求自行配置环境变量。

Fork 并克隆本仓库，安装依赖：

```sh
npm i
```

然后进行修改。你可以使用以下命令实时预览：

```sh
npm run docs:dev
```

修改完成后，**请务必在本地执行一次构建，以确认不会产生构建错误**：

```sh
npm run docs:build
```

提出 PR 后若长时间无人审核，可前往[社区](https://awavenue.top/Support.html#%E7%94%A8%E6%88%B7%E4%BA%A4%E6%B5%81)提出。
